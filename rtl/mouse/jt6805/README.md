## Device differences

Item           | MC6801            | MC6805          | HD6301
---------------|-------------------|-----------------|-------
Type           | MCU               | MCU             | MCU
B              | present           | absent
X/S            | 16 bits           | 8 bits
Idx addr.      | 8-bit offset      | 8/16-bit offset
